# Slogan GitHub Pages 上传版 V2

直接上传本文件夹内所有内容到仓库根目录，开启 Settings - Pages - main / root。

本版已加入像素画素材和东方水墨素材。

## 首页图片修复说明
本版本已将首页主视觉从单纯 CSS background 改为真实 `<img>` 标签加载，并保留 CSS 背景兜底，适合 GitHub Pages 直接部署。


## 互动优化
本版本新增滚动进度、移动端菜单、浮动光标、滚动显现、卡片悬停、色卡复制、构图辅助线切换和响应式布局。


## 站头升级
首页已加入新版大气站头视觉，文件位于 assets/images/hero-keyvisual.jpg，并保留完整概念图 assets/images/hero-concept-full.jpg。


参考网站优化说明：根据同学网站的长处新增“进入视觉档案”入口、四模块路径、线索索引条和视觉气质测试，提升首页引导、互动感与内容层次。
